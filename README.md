# Introduce
Inject  shellcode in to a PE file 

# USE
Please make sure your machine has `Mingw-w64` `MAKE` `NASM` and `GOLANG` installed . If your machine OS is windows , you need install `bash cmd`.

On terminal enter:
```
    FILE=./PE.exe make
```
